Hugo基础

zzl 18 October, 2019 share

1.Hugo的优点

  • 静态文件，安全、备份方便；
  • 所有页面数据源头都是一个md文件，纯文本（用什么软件都可以打开）、格式统一、管理方便；
  • 安装方便，不需要依赖复杂的运行环境；

2.安装Hugo

  1. 下载二进制安装包，地址：https://github.com/gohugoio/hugo/releases
  2. 创建一个新文件夹：C:\Hugo，你可以在任何地方创建，这里将Hugo放在C盘；
  3. 在Hugo文件夹中创建一个子文件夹： C:\Hugo\bin；
  4. 在Hugo中创建另一个子文件夹： C:\Hugo\Sites；
  5. 将下载的安装包内容解压到C:\Hugo\bin；
  6. 将C:\Hugo\bin添加到你的环境变量Path;

3.创建一个基础站点

  1. 进入C:\Hugo\Sites，运行hugo new site 你的项目名称；

4.进一步配置：标题、副标题、侧边栏等

```
# 基础设置
baseURL = "http://balaramadurai.github.io/" #生成网站之后替换这地址
title = "Hugo" #网站标题
subtitle = "Hugo基础使用文档" #网站副标题
languageCode = "zh-cn" #语言，取决于主题的设置，这里设置了好像没用
theme = "hugo-travelify-theme" #主题
GoogleAnalytics = "" # 添加你的谷歌分析代码
canonifyURLs = true

# 为右边栏启用和禁用小部件
[params.widgets]
    recent_articles = true # 是否显示最近打开过的文档
    categories = true # 是否显示分类模块
    tags = true # 是否显示标签模块
    tag_cloud = true
    search = false # 是否显示搜索模块
    mailchimp = false
    mailchimpform = "//balaramadurai.us15.list-manage.com/subscribe/post?u=7f4183fa2fe7abe3dc9f4efad&amp;id=49a3dd3745"
    author = true # 是否显示作者模块

[params.slider]
    enable = true   # 是否显示轮播图模块
    manual = false   # Setting manual to true renders the slide selection to only those where the post has a slider=true option enabled
                    # 设置为true，则只会选择启用了slider=true选项的帖子呈现幻灯片
    num_features = 5 # The slider picks up the banner images from the N most recent posts, where num_features is N.
                    # 滑块从最近的N篇文章中获取横幅图像，前提是manual = false
[Taxonomies]
    tags = "tags"
    categories = "categories"
    author = "author"
```

5.使用hugo创建一篇文章

  1. 创键一篇文章命令：hugo new 文章的路径/文章的名称

    PS C:\Hugo\Sites\hugoShare> hugo new post/hugo-share.md
C:\Hugo\Sites\hugoShare\content\post\hugo-share.md created

  2. 新建的文档自带一段文件头（ front matter）：

    ---
title: "Hugo Share"
date: 2019-10-18T17:18:42+08:00
draft: true
---

    title：表示这个文档的标题；

    date：表示这个文档创建的时间，这个时间的显示格式，可以在config.toml里配置date_format参数；

    draft：表示这个文档是否是草稿，值为true是草稿，草稿的文档不会生成；

    除此之外，你还可以添加其他信息：

    author: "zzl" # 作者
tags: ["hugo"] # 所属标签
categories: ["share"] # 所属分类
menu: ""
banner: "banners/7148951717_9bbf185db3_h-1018x460.jpg" # banner图地址
slider: true # 是否显示在轮播图里，与config.toml文件里的[params.slider]参数配合使用
disable_profile: true # 是否显示侧边栏

    具体参见 https://gohugo.io/content-management/front-matter/ 和主题的使用说明

  3. 完善你的文章内容；

  4. 生成静态文件：生成的命令就是：hugo

    PS C:\Hugo\Sites\hugoShare> hugo
